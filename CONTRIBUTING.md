# Contribuir a PlaceNote-Server

Gracias por tu interés. PlaceNote es un proyecto **open source** y self-hosted; las contribuciones son bienvenidas.

## Cómo empezar

1. Haz fork del repositorio y crea una rama desde `main`.
2. Asegúrate de que `./gradlew build` pasa antes de abrir un pull request.
3. Si cambias el contrato HTTP, actualiza **siempre** [docs/api/openapi.yaml](docs/api/openapi.yaml) y menciona en el PR el impacto en [PlaceNote-Client](https://github.com/AlvaroMinarro/PlaceNote-Client).

## Estilo

- Kotlin idiomático, nombres en inglés para código y APIs públicas salvo decisión explícita del proyecto.
- Commits con mensajes claros en una sola línea o cuerpo explicativo cuando haga falta.

## Reportes

Usa los issues de GitHub para bugs y propuestas; indica versión del servidor y pasos para reproducir.
